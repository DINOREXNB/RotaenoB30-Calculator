## 先决条件和部署

1. 安装 [Node.js](https://nodejs.org)
2. 打开 PowerShell/终端，导航到当前目录，然后输入`npm install`来安装依赖项。
3. 在 PowerShell/终端中输入`npm start`来启动服务器。
4. 在 `http://localhost:9876/RotaenoB30.html` 输入您的实际rating。
